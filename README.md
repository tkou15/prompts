# prompts

AI 開発ツール（GitHub Copilot, Cursor, ChatGPT 等）のプロンプトテンプレートとシステムインストラクション集。プロジェクトごとのベストプラクティスを一元管理するためのナレッジベース。

## ディレクトリ構造

```
├── github-copilot/
│   ├── general/            # 全般的なルール
│   ├── frontend/           # フロントエンド特化
│   └── backend/            # バックエンド特化
└── claude/                 # Claude用のskills
```
